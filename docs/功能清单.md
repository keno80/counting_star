# 记账软件功能清单

## 1. 账本与账户

- 账本管理：新建/编辑/删除/归档账本；多账本切换；默认账本
- 账户管理：现金/银行卡/信用卡/第三方钱包/借贷账户；账户启用/停用
- 账户属性：币种、初始余额、账单日/还款日（信用卡）、额度、利率（可选）
- 账户操作：账户间转账；余额调整；冻结/解冻（可选）

## 2. 分类、标签与商家

- 收入/支出分类：两级分类；排序；常用置顶；合并/迁移分类
- 标签：多标签；标签颜色/图标（可选）；按标签统计
- 商家/交易对象：常用商家；历史自动联想；按商家统计（可选）

## 3. 记账（流水）

- 记一笔：收入/支出/转账；金额；账户；分类；日期时间；备注；标签；商家
- 附件：图片/票据（可选）；多附件；预览与删除
- 拆分记账：一笔拆多分类/多标签/多金额
- 退款/冲正：关联原流水；自动反向入账（可选）
- 分摊：多成员分摊；AA/自定义比例；待结算清单（可选）
- 批量操作：批量编辑分类/标签/账户；批量删除；批量导出
- 复制/模板：复制上一笔；快捷模板；常用金额/分类快捷按钮（可选）

## 4. 流水列表与查询

- 列表展示：按时间分组；日/周/月视图（可选）；滚动加载
- 搜索：关键字（备注/商家/标签）；金额区间；日期区间
- 筛选：账户/分类/标签/成员（如有）/是否有附件
- 排序：时间/金额；正序/倒序
- 详情页：查看/编辑/删除；关联信息（转账两端、退款关系等）

## 5. 预算与提醒

- 预算：按月/按周；按分类/总预算；滚动预算（可选）
- 预算进度：剩余/超支提醒；按天建议可用额度（可选）
- 提醒：账单日/还款日提醒；定时记账提醒（可选）

## 6. 统计与报表

- 概览：本月收入/支出/结余；同比/环比（可选）
- 分类统计：饼图/条形图；Top N；自定义时间范围
- 账户统计：账户余额趋势；净资产（资产-负债）；现金流（可选）
- 标签/商家统计：标签维度/商家维度聚合（可选）
- 报表导出：CSV/Excel（可选）；生成分享图片（可选）

## 7. 定时与自动化（可选增强）

- 定时记账：按天/周/月；跳过规则（节假日等可选）
- 自动归类：基于商家/备注关键词规则；手动纠错学习（可选）
- 快捷入口：桌面小组件/快捷指令（可选）

## 8. 数据导入导出与备份

- 导入：支付宝/微信/银行流水/CSV 自定义映射（可选）
- 导出：全量导出；按时间/账户导出；脱敏导出（可选）
- 备份：本地备份文件；自动备份；恢复与冲突处理

## 9. 多端与同步（可选增强）

- 登录：手机号/邮箱/第三方登录（按项目技术选型）
- 同步：多设备实时/准实时同步；离线可用；冲突合并策略
- 多账本协作：共享账本；成员角色（管理员/编辑/只读）（可选）

## 10. 设置与个性化

- 记账偏好：默认账户/分类；小数位；周起始日；时间格式
- 主题与显示：深色模式；图标/配色（可选）；字体大小（可选）
- 分类/标签管理入口：导入导出分类；一键整理（可选）

## 11. 安全与合规（可选增强）

- 隐私：本地密码/生物识别解锁（可选）
- 数据安全：加密存储（可选）；敏感信息脱敏展示（可选）
- 审计：操作日志（可选）

## 12. 非功能性需求（建议）

- 性能：列表加载与统计计算响应时间目标；大数据量（10万笔）可用性（可选）
- 可靠性：崩溃恢复；备份可用；导入幂等（重复导入可识别）
- 可用性：误删回收站（可选）；撤销（可选）；空状态与引导
- 可测试性：核心计算（结余/净资产/预算）可单测；导入映射可回归

## 13. MVP 建议范围（第一版可交付）

- 账本/账户/分类/标签基础管理
- 记一笔（收入/支出/转账）+ 列表查询 + 编辑删除
- 月度概览 + 分类统计（基础）
- 导出（CSV）+ 本地备份与恢复
